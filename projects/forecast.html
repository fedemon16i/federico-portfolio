What I Owned

Scope of ownership

Led end-to-end UX architecture for a workforce management platform at Applaudo Studios
Designed 4 modular capability areas covering scheduling, tracking, reporting, and allocation
Built a unified design system enabling consistent UI across all modules
Validated designs through iterative user testing with internal stakeholders
Approach

How I worked

01 — Discovery

Map internal workflows and pain points

Ran structured stakeholder interviews to understand existing workforce management workflows, surface friction points, and align on platform scope before any design.

02 — Architecture

Structure modular system to support 5 capability areas

Designed a modular information architecture that allowed each of the 5 capability areas to function independently while sharing a coherent navigation and interaction model.

03 — Design System

Build once, apply across all modules for consistency and speed

Established a shared component library and token system in Figma — a single design system deployed across all 4 modules, reducing duplication and accelerating handoff.

04 — Validate

Iterative testing with real users before handoff

Conducted iterative usability sessions with internal Applaudo users to refine flows, resolve navigation ambiguities, and confirm the platform met real operational needs.
