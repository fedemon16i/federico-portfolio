Design Systems & Prototype

Two Design Systems. One Spatial Vision.

Web UI Layer — Flat DS
Black & Neon

The platform's web-facing UI used a flat design system — bold black backgrounds, neon yellow (#CBFF00) as the primary action color. Designed in Figma, applied across all 2D interactive surfaces.

Platform Architecture

The Platform Also Lives on the Web

Beyond the mobile app, the platform required a full web companion — a management dashboard where Space owners configure their 3D environments, manage POIs, and monitor activity. Designing for both mobile and desktop surfaces demanded strict design system coherence.

Space Dashboard

Central hub for managing multiple 3D Spaces — visitor analytics, content calendar, and POI health at a glance.

3D Map Web

Full three.js map view on desktop — drag, zoom, and click-to-edit POIs directly in the browser without Unity.

3D Asset Library

Searchable catalogue of Unity-ready 3D objects. Owners browse, preview, and attach assets to any POI from the web dashboard.

Multi-role Access

Owner, Editor, and Viewer roles — each with distinct permissions. Designed so the same component set serves all three without diverging UI branches.
